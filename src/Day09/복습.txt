[ 생성자 ]
1. 정의 : 객체(인스턴스)를 생성할 때 사용되는 초기화 메소드
    예) 클래스명 변수명 = new 생성자명();
2. 규칙
    1) 클래스명과 생성자명은 동일하다
    -> 다르면 메소드 취급
    2) 오버로드 : 매개변수의 개수/타입/순서가 다르면 동일한 이름으로 여러개 선언하는 것
    3) 메소드와 다르게, 반환값이 없음
3. 종류
    1) 기본 생성자 : 클래스 내 생성자가 하나도 존재하지 않으면, 컴파일될 때 자동 생성
        -> 매개변수가 없는 생성자
    2) 매개변수가 있는 생성자
4. 목적 : 객체 생성 시, 멤버변수의 빠른 초기화를 위해
5. 사용법
    1) 선언 방법
    -> 생성자명( ){ }
    2) 호출 방법
    -> 클래스명 변수명 = new 생성자명( );
